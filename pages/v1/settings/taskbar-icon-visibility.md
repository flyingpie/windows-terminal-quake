# Taskbar Icon Visibility

When to show the terminal window icon on the taskbar.

"AlwaysHidden", "AlwaysVisible" or "WhenTerminalVisible".

Defaults to no "AlwaysHidden".

```json
{
	"TaskbarIconVisibility": "AlwaysHidden"
}
```
